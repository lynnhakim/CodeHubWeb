The Code Hub C++ Lessons

Posted by Lynn, January 31, 2023

C++ is a powerful and versatile programming language. Known for its efficiency and object-oriented features, C++ facilitates both low-level memory manipulation and high-level abstraction.

Variables / Assignments

  • Data Types 3 practice problems

      • Discover the different data types that are used in C++ such as Booleans, integers, strings, floats, and others.

  • Operators 2 practice problems

      • Learn how operators are used to assign values to variables. They range from arithmetic, relational, and logical.

Branches

  • Conditional Statements 2 practice problems

      • Construct if-else statements to allow execution of specific code blocks based on Boolean expressions.

  • Switch Statements 1 practice problem

      • A concise and efficient way to handle multiple cases. This enhances code readability in scenarios with multiple conditional paths.

Vectors

Declaration Description
vector<int> amounts; Declares amounts as an empty vector of ints.
vector<int> scores(15); Declares scores as a vector of 15 ints.
vector<char> letters(25, 'A'); Declares letters as a vector of 25 characters. Each element is initialized with 'A'.
vector<double> values2(values1); Declares values2 as a vector of doubles. All the elements of values1, which is also a vector of doubles, are copied to values2.

*Number of practice problems is subject to change
